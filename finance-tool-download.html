📊 Finance Tool — Full Source Code

Sales & Returns reporting tool. Connects to PostgreSQL and/or MSSQL. All 20 source files are embedded below.

Step 1 — Configure DBRename backend/.env.example to backend/.env and fill in your PostgreSQL / MSSQL credentials.
Step 2 — Map your columnsOpen backend/routes/sales.js and routes/returns.js — edit TABLE_CONFIG at the top with your real table/column names.
Step 3 — Install & buildcd backend && npm install
cd ../frontend && npm install && npm run build
Step 4 — Runcd backend && node server.js
Share http://YOUR-SERVER-IP:4000 with finance team.

Set API_SECRET in .env as the finance team login token. Set SALES_DB / RETURNS_DB to postgres or mssql per report.
